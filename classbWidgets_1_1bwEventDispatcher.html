Detailed Description

Mangages sending events to screen-graph nodes based on current state.

Based on the screen-graph context as state, the event dispatcher sends events to the screen-graph nodes in user focus (i.e. hovered or active node). That means, it calls the nodes event listener corresponding to the determined event.
